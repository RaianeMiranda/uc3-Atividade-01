Descrição de Linguagens

Para que serve o HTML?

A linguagem de HTML(Hyper Text Markup Language) permite a construção de websites com inserção de imagens, vídeos, links e outros atributos.

O html pode ser usado em:

  • Parte visual de aplicações web, como no frontend;
  • Desenvolvimento de aplicações Mobile para Android e iOS;
  • Desenvolvimento de jogos

Para quê serve o CSS?

A linguagem de CSS(Cascating Style Sheets) é uma linguagem de marcação usado juntamente ao HTML, ou seja, é um código utilizado na representação visual dos Websites.

Para quê serve o Markdown?

O Markdown é um sistema de formatação simples e aberta, sendo ele visualmente limpo e pode ser facilmente convertido em HTML. Basicamente as alterações nos textos são feitos apenas com os símbolos do teclado, sem um um visual complexo como os de HTML ou CSS.
